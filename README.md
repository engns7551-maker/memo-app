# Memo App

HTML, CSS, JavaScript로 만든 간단한 메모장 웹앱입니다.

메모를 작성하고, 수정하고, 삭제할 수 있으며 태그와 검색 기능도 사용할 수 있습니다. 작성한 메모는 `localStorage`에 저장되어 브라우저를 새로고침해도 데이터가 유지됩니다.

## Live Demo

https://engns7551-maker.github.io/memo-app/

## 주요 기능

- 메모 작성
- 메모 수정
- 메모 삭제
- 태그 입력 및 표시
- 제목, 내용, 태그 검색
- 작성일 표시
- `localStorage`를 이용한 새로고침 후 데이터 유지

## 사용 기술

- HTML
- CSS
- JavaScript
- localStorage
- Git / GitHub

## 실행 방법

1. 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 메모를 작성하고 기능을 확인합니다.

별도의 서버나 설치 과정 없이 브라우저에서 바로 실행할 수 있습니다.
